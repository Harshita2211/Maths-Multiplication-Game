# Maths Multiplication Game

A simple and interactive multiplication practice game built using **HTML, CSS, and JavaScript**. Players answer random multiplication questions and try to score as many points as possible within the time limit.

##  Features
- Random multiplication questions  
- Timer-based gameplay  
- Instant answer validation  
- Score tracking  
- Clean and responsive UI  

##  Project Structure
```
/project
│── index.html
│── style.css
│── script.js
```

##  How to Run
1. Download or clone the project.  
2. Open **index.html** in any modern web browser.  
3. Start playing immediately—no installation required!

##  How It Works
- A random multiplication problem is generated on screen.  
- The user enters an answer in the input box.  
- JavaScript checks the answer and updates the score.  
- When the timer ends, the final score is displayed.

##  Technologies Used
- **HTML5** – game layout  
- **CSS3** – styling and responsiveness  
- **JavaScript** – game logic and interactivity
